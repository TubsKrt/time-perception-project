# Time Perception Analysis

This project simulates a psychological experiment on time perception.

## What it does
- Compares real vs perceived time
- Calculates time distortion
- Computes average bias

## Psychology relevance
Time perception is often biased. This project shows how perceived time differs from real time.

## Technologies
- Python
- Pandas

## Example Output

Average distortion: 3.2

## Psychological Insight

People tend to overestimate time when stressed and underestimate it when relaxed.
This project simulates that cognitive bias.
